# Bytes Calculator using Join/Fork methods

This folder has a program that calculates the amount of bytes present in a directory. 

## Results 

```
---------------------------------------------------------
|~ The conventional sum :
| on C:\temp directory
| -> Result = 9455239934 Bytes
| -------------------------------------------------------
|~ Runtime: 1349 ms
---------------------------------------------------------
---------------------------------------------------------
|~ The sum with join/fork method:
| on C:\temp directory
| -> Result = 9455239934 Bytes
| -------------------------------------------------------
|~ Runtime: 395 ms
---------------------------------------------------------
```
